# Bott
A very simple Slack Bot

# Tools
* Virtualenv
* Free Slack account with a team on which you have API access
* Official Python slackclient code library

# Variables 
* export SLACK_BOT_TOKEN = 'Your slack token here'
* export BOT_ID = 'Your Bot Id'
